מנחם לימוזין - אתר הסעות VIP
אתר רשמי לחברת הסעות VIP מנחם לימוזין. האתר מציג את השירותים, צי הרכבים, ומאפשר יצירת קשר מהירה עם החברה.

תכונות האתר
עיצוב מודרני, נקי ורספונסיבי המותאם לכל המכשירים

תמיכה מלאה בעברית ובכיווניות RTL

תצוגת שירותים מתקדמת ופירוט מלא של צי הרכבים

טופס יצירת קשר ואמצעינגשות מתקדמים

תמיכה במדיה חברתית ובקישורים ישירים

דרישות מערכת
דפדפן מודרני עם תמיכה ב-CSS3 ו-JavaScript ES6

חיבור לאינטרנט (לצורך טעינת פונטים, אייקונים וספריות חיצוניות)

מבנה קבצים והתקנה
ודא שכל הקבצים המעודכנים נמצאים ישירות בתיקיית השורש (Root) של הפרויקט בלי תיקיות ביניים:

index.html

styles.css

script.js

קובצי עזר נוספים (accessibility.html, cookies.html, privacy.html, terms.html וכו')

תיקיית images/ (הכוללת את תמונות הרכבים והרקע)
